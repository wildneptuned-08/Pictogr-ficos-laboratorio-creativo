# Visión del Negocio

## Empresa

Negocio de artes gráficas dedicado a sublimación, impresión y diseño.

## Problema actual

La operación depende de Google Sheets y WhatsApp Business, lo que puede
ocasionar pérdida de pedidos y poca trazabilidad.

## Objetivo V1

Centralizar pedidos, clientes, finanzas, inventario y presupuesto en una
aplicación web moderna.

## Flujo principal

Cliente → Pedido → Producción → Entrega → Pago → Distribución financiera
→ Dashboard.
