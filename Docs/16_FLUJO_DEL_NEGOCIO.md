# Flujo del Negocio

## Objetivo

Documentar el proceso completo desde que un cliente solicita un producto
hasta el cierre financiero.

## Flujo Principal

1.  Cliente contacta por WhatsApp.
2.  Se registra el pedido.
3.  Se define producto y diseño.
4.  Se registra anticipo (si aplica).
5.  Se agenda producción.
6.  Se entrega el producto.
7.  Se registra el pago final.
8.  El sistema calcula utilidad.
9.  Se distribuye automáticamente entre los bolsillos.
10. El Dashboard se actualiza.

## Regla

Todo pedido debe tener trazabilidad.
