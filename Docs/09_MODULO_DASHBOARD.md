# Módulo Dashboard

## Objetivo

Mostrar en tiempo real el estado del negocio.

## KPIs

-   Ventas del día, semana y mes
-   Pedidos pendientes
-   Pedidos entregados
-   Utilidad
-   Cumplimiento de metas
-   Alertas de inventario

## Reglas

Toda la información proviene de Supabase. No almacenar cálculos
permanentes.

## Criterios de aceptación

Los indicadores deben actualizarse automáticamente al registrar cambios.
