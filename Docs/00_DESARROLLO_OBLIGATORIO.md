Construir una V1 profesional, modular y escalable.

-   No romper funcionalidades aprobadas.
-   Arquitectura modular.
-   TypeScript obligatorio.

-   Responsive.
-   Variables de entorno.
-   No duplicar código.
-   Documentar cambios.
